# Проект для управления базой данных

Этот проект представляет собой приложение для управления базой данных с использованием SQLite. Он позволяет пользователям создавать проекты, добавлять навыки к проектам и отслеживать статусы проектов.

## Функциональные возможности

- Создание таблиц для проектов, навыков и статусов.
- Добавление и извлечение данных из базы данных.
- Поддержка различных статусов проектов.
- Возможность добавления фото к проектам.

## Используемые технологии

- Python
- SQLite
- API
